# UniPulse
Веб-сайт для навигации по территории и корпусам Московского авиационного института, помогающее быстро находить аудитории, кафедры и объекты инфраструктуры.
## Команда

- Маляренко Елизавета: Дизайнер.
- Ерзюкова Вероника: Frontend-разработчик (сайт).
- Савошинский Алексей: Backend-разработчик (сервер).
- Швецов Иван: Тестировщик.
## Цель проекта
Разработать прототип картографического сервиса, который позволяет пользователям кампуса быстро получать маршруты и информацию о точках интереса, решая проблему отсутствия единой навигационной системы МАИ.
## Технологии
- Frontend: HTML, CSS, JavaScript
- Backend: Python, Flask
- База данных: SQLite
- Инструменты: GitHub, Figma
